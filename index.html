Experiências Personalizadas

O S.U.M oferece interfaces específicas para cada tipo de usuário

Aluno

Os alunos têm acesso a:

  • Notas e desempenho
  • Sala de aula virtual
  • Atividades e provas
  • Perfil DOTS completo
  • Chat com professores

Professor

Os professores podem:

  • Gerenciar salas de aula
  • Criar atividades com IA
  • Mapeamento inteligente
  • Acompanhar desempenho
  • Comunicar-se com alunos

Escola

As escolas têm controle sobre:

  • Cadastro de usuários
  • Gestão de turmas
  • Relatórios completos
  • Sistema de gerenciamento
  • Personalização do sistema
