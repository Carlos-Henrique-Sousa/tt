Estrutura Básica dos Arquivos HTML

Padrão Comum

Todos os arquivos HTML seguirão esta estrutura básica, importando os mesmos arquivos CSS e JS para manter a consistência visual e funcional:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nome da Página - S.U.M</title>
    <link rel="stylesheet" href="style.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
</head>
<body>
    <header>
        <!-- Cabeçalho com navegação -->
    </header>
    
    <main>
        <!-- Conteúdo específico da página -->
    </main>
    
    <footer>
        <!-- Rodapé -->
    </footer>
    
    <script src="script.js"></script>
</body>
</html>

Observações Importantes

  • Todas as páginas compartilharão o mesmo cabeçalho e rodapé para consistência
  • O arquivo style.css conterá todos os estilos do sistema
  • O arquivo script.js conterá a lógica JavaScript comum
  • Usaremos Three.js para animações em páginas específicas (mapeamento, dashboard)
  • A navegação entre páginas será feita através de links tradicionais
  • As páginas marcadas como "Alta" prioridade serão desenvolvidas primeiro
  • Esta estrutura pode ser ajustada conforme o desenvolvimento avança
